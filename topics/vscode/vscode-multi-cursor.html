Ctrl+D 系ショートカット徹底解説

複数カーソル（Multi-cursor）を自在に扱うための完全ガイド

Ctrl+D とは何か

VSCode の Ctrl+D は、 カーソル位置の単語と同じ文字列を、1 つずつ順番に複数カーソルで追加選択する ショートカットです。

一度に選択するのではなく、押すたびに次の一致を増やしていくのが特徴。これにより「この 3 箇所だけ」「ここは除外して次」のような細かい制御ができます。

複数カーソルとは: 同時に複数位置でタイピング・削除・選択ができる編集モード。Ctrl+D はその最もよく使う入り口です。

基本動作を 4 ステップで見る

変数 user を currentUser にリネームする流れを追ってみます。

カーソル位置 選択中の文字列 入力で書き換わった箇所
0
開始状態：カーソルが user の中にある
まだ何も押していない
const user = getUser();
console.log(user);
return user.name;
1
1 回目：単語が選択される
Ctrl+D
const user = getUser();
console.log(user);
return user.name;

最初の 1 回は単語選択のみ。カーソル数は変わらず 1 つ。

2
2 回目：次の一致にカーソル追加
Ctrl+D を再度
const user = getUser();
console.log(user);
return user.name;

2 箇所が同時選択された状態。カーソルが 2 つになる。

3
3 回目：3 箇所目を追加
Ctrl+D ×3
const user = getUser();
console.log(user);
return user.name;
4
タイピングで同時書き換え
currentUser と入力
const currentUser = getUser();
console.log(currentUser);
return currentUser.name;

3 箇所すべてが一度のタイピングで書き換わる。Esc で複数カーソル解除。

Ctrl+D ファミリー一覧

Ctrl+D を軸に、関連するショートカットを覚えるとほぼすべての複数選択ユースケースをカバーできます。

ショートカット動作使い所
Ctrl+D 次の一致にカーソルを追加 1 箇所ずつ確認しながら広げたいとき
Ctrl+Shift+L 同一文字列をすべて一発選択 ファイル全体で確実に同じ単語を置換
Ctrl+K,Ctrl+D 現在の選択をスキップして次へ 「ここは変えたくない」箇所を飛ばす
Ctrl+U 直前に追加したカーソルを取り消す Ctrl+D を押しすぎた時の「1 歩戻る」
Alt+クリック クリック位置に手動でカーソル追加 関係ない行にも散らばせたいとき
Ctrl+Alt+↑ / ↓ 上下に縦方向のカーソル追加 縦に並んだ値・列の一括編集
Shift+Alt+ドラッグ 矩形（ボックス）選択 CSV / テーブル状データの列編集
Esc 複数カーソルを解除 → 単一カーソルに戻す 編集が終わったら忘れずに

Ctrl+D vs Ctrl+Shift+L vs Ctrl+H の使い分け

同じ単語を置換する手段は 3 つあり、それぞれ得意領域が違います。

Ctrl+D

段階選択

1 箇所ずつ追加。目で見て確認しながら選べる。

範囲: 手動で拡大
Ctrl+Shift+L

全選択

ファイル内の一致を一瞬ですべて選ぶ。確実さ重視。

範囲: ファイル全体
Ctrl+H

検索・置換

正規表現・大文字小文字・単語単位などを細かく制御可能。

範囲: 条件で指定
やりたいこと推奨
関数内の user 3 箇所だけ変えたい Ctrl+D ×3
ファイル内の userId を 全部 accountId に Ctrl+Shift+L
正規表現で log\(.*\) を無効化 Ctrl+H（正規表現モード）
プロジェクト全体で文字列を置換 Ctrl+Shift+H

実用パターン 3 選

① 変数リネーム（スコープ限定）

LSP 経由の正式リネーム（F2）より軽量で、関数内だけといった狭い範囲に使いたい時に便利です。

// user → currentUser（この関数内だけ）
function greet(user) {
  return `Hello, ${user.name}`;
}

② インポート文の一括編集

パッケージ名変更やエイリアス変換に。1 行ずつ変えるより速い。

import { a } from "@old/core";
import { b } from "@old/core/utils";
import { c } from "@old/core/types";

③ 縦に並んだ値の一括編集（Ctrl+Alt+↓）

.env ファイルや設定オブジェクトの値を、縦方向に複数カーソルで一気に追加・編集できます。

DATABASE_URL= 
REDIS_URL= 
API_KEY= 

行末で Ctrl+Alt+↓ を連打 → 3 行目まで縦にカーソルを伸ばす → まとめてペースト。

ハマりどころと対処

Ctrl+D を押しすぎた: 余分なカーソルを Ctrl+U で 1 つずつ戻せます。連打で複数回戻すのも可。
意図しない箇所まで選択された: その状態で Ctrl+K,Ctrl+D を押すと、今の選択をスキップして次へ移ります。
部分一致してしまう: user を選ぶと username の中の user も拾います。単語境界で止めたい場合は、検索ウィジェット（Ctrl+F）の「単語単位で検索」（Match Whole Word）を併用すると安全です。
複数カーソルから抜けたい: 編集が終わったら Esc で単一カーソルに戻すクセをつけると、意図しない挙動を防げます。

まとめ

  • Ctrl+D = 次の一致を 1 つずつ追加（段階的）
  • Ctrl+Shift+L = 一発で全部選択（ファイル内）
  • Ctrl+U = やりすぎたら 1 歩戻す
  • Ctrl+K, Ctrl+D = 今の選択はスキップして次へ
  • Ctrl+Alt+↑/↓ = 縦方向のカーソルで列編集
  • Esc = 複数カーソルを解除

この 6 つを覚えれば、コードのリネームや一括編集のほとんどの場面で「マウスで範囲選択 → コピペ」をせずに済みます。
