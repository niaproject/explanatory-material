# explanatory-material

技術トピックに関する解説資料をまとめたリポジトリです。各種テーマごとにモダンなHTMLで可視化・解説したページを収録しています。

## 収録コンテンツ

| カテゴリ | ファイル | 内容 |
| --- | --- | --- |
| Git | [git/git-commands.html](git/git-commands.html) | Git の主要コマンド（add / commit / push / HEAD など）の関係性を図解 |
| Git | [git/git-unstage.html](git/git-unstage.html) | `git add` を取り消す手順（特定ファイルだけ / 全部）を図解 |
| Hosting | [hosting/hosting-comparison.html](hosting/hosting-comparison.html) | 各種ホスティングサービスの比較解説 |

## 使い方

各 HTML ファイルをブラウザで開くと、解説ページが表示されます。

## ディレクトリ構成

```
explanatory-material/
├── git/         Git 関連の解説資料
└── hosting/     ホスティング関連の解説資料
```
